The Architecture of Aura

Aura is an advanced Sentiment Engine combining sleek modern web design with powerful open-source Natural Language Processing libraries, hosted locally and synchronized with Google Cloud infrastructure.

Presentation Layer
Vanilla JS + Tailwind CSS
API & Auth Layer
FastAPI | Firebase Auth
NLP Engine
TextBlob, spaCy,
NRCLex
Database
Cloud Firestore

Frontend Technology

  • •
    Vanilla JavaScript (ES6+): Built without heavy React or Vue frameworks for ultimate speed, leveraging raw DOM manipulation.
  • •
    Tailwind CSS: Utility-first styling enabling the highly custom, glassmorphic dark-theme design.
  • •
    Chart.js: Used to render the real-time "Personal Trend" dashboard, mapping sentiment confidence curves.

Backend Server

  • •
    Python 3 & FastAPI: A highly performant, asynchronous web framework that easily handles bulk-upload API requests.
  • •
    Firebase Admin SDK: Python library used to verify OAuth Bearer tokens, rejecting unauthorized/spoofed requests.
  • •
    Cloud Firestore DB: A NoSQL document database used over SQLite to allow for highly scalable, real-time query saving.

Natural Language Processing Pipeline

1. Polarity Analysis

Utilizes TextBlob to mathematically determine if text is Positive, Negative, or Neutral.

2. Emotion Extraction

Powered by NRCLex, extracting raw underlying emotions (e.g. Trust, Anger, Joy, Fear).

3. Aspect-Based Sentiments

Uses spaCy dependency parsing to isolate nouns (aspects) and determine their specific targeted sentiment.

4. Named Entity Recognition

The en_core_web_sm ML model identifies dates, people, organizations, and geographical places.
