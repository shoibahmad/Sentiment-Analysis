Aura API v2.0

Welcome to the Aura Developer Portal. Our API provides advanced sentiment intelligence, emotion extraction, and dark pattern detection for your applications.

📡

Base URL

http://localhost:8000/api

Authentication

Aura uses Firebase Authentication. All requests must include a valid ID Token in the Authorization header using the Bearer scheme. 

Authorization: Bearer <YOUR_FIREBASE_ID_TOKEN>
POST /analyze

Analyze Single Text

Processes a single string of text through the full NLP pipeline, including sentiment, emotions, Named Entity Recognition (NER), and manipulation detection.

Request Body

{
  "text": "Per my last email, I think we should leverage our synergy."
}

Example cURL

curl -X POST http://localhost:8000/api/analyze \
  -H "Authorization: Bearer TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello world"}'
POST /analyze/bulk

Bulk File Analysis

Upload a file containing multiple texts for batch processing. Gemini AI will generate an executive summary based on the collective results.

Form Data

Key Type Description
file file Supported: .csv (with 'text' column) or .txt

Interactive Summary Response

{
  "processed_count": 50,
  "executive_summary": "Most users find the interface intuitive...",
  "summary": { "Positive": 40, "Negative": 5, "Neutral": 5 },
  "results": [...]
}

🌀 Manipulation Detection

A unique feature that identifies dark patterns, gaslighting, and passive-aggressive behavior. The `manipulation` object in the response gives you high-level insights.

  • score 0.0 to 1.0 (Higher means more manipulative)
  • label "Sincere Content", "Highly Manipulative", etc.
  • clues Array of detected patterns (e.g., "Passive-aggressive markers")
