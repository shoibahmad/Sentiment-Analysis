About The Project

Built for speed, accuracy, and aesthetics.

Aura is a completely localized, full-stack machine learning engine. It seamlessly bridges a high-performance Python FastAPI backend with a zero-dependency Vanilla JavaScript frontend.

We go beyond simple polarity matching. Utilizing `spaCy` for Named Entity Recognition (NER), `TextBlob` for cross-lingual sentiment base lining, and `NRCLex` for raw emotive extraction, Aura paints a full picture of the data you feed it.

// core_engine.py
from nrclex import NRCLex
import spacy
# Advanced multi-metric analysis
nlp = spacy.load("en_core_web_sm")
doc = nlp(text)
entities = [ent.text for ent in doc.ents]
# Returns { "fear": 0.5, "joy": 0.8 ... }
return NRCLex(text).raw_emotion_scores
